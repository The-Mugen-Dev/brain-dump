## Init commands
- [ ] ```bash xcode-select --install```
- [ ] Add git ssh
- [ ] Install xcode if needed
- [ ] Install Home Brew -> [Homebrew](https://docs.brew.sh/Installation)
- [ ] Install iTerm2 -> ```bash brew install --cask iterm2```
- [ ] Install ZSH -> ```bash brew install zsh```
- [ ] Install OhMyZsh -> ```bash sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"```
- [ ] Update theme to af-magic on ~/.zshrc and increase contrast
