## Init commands
- Installing the Xcode Command Line Tools
```bash
xcode-select --install
```
- Add [Git SSH](https://docs.github.com/es/authentication/connecting-to-github-with-ssh)
- Install [Homebrew](https://docs.brew.sh/Installation)
- Install iTerm2
```bash
brew install --cask iterm2
```
- Install ZSH
```bash
brew install zsh
```
- Install OhMyZsh
```bash
sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"
```
- Update OhMyZsh theme to af-magic on `~/.zshrc` and increase minimum background contrast in terminal preference
- If using Idea products disable "use non-modal commit interface" under Preferences
